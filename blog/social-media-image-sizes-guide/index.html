A Complete Guide On the Social Media Image Sizes

Whether it is your LinkedIn banner or Facebook profile picture, they are considered to be the ultimate backbone of every social media platform. If your images are compelling, you will taste victory in the engagement battle. It is an essential tool to encourage your customers to visit your profile. This guide will cover all the required social media image sizes to get the perfect visuals.

Facebook Image Sizes

An effective way to grab the attention of Facebook users is to have top-quality images. You can use Facebook images to improve your customer engagement. But, since Facebook continuously updates the design and dimensions of ideas, you need to ensure you upload great photos on Facebook. So, below, we have provided some of the recommended file formats and image dimensions.

Facebook Profile Image Size

  • For Desktop- 170 x 170 pixels
  • For Mobile- 128 x 128 pixels

Facebook Cover Photo Size

  • For Desktop- 820 x 312 pixels
  • For Mobile- 640 x 360 pixels
  • Recommended Size- 851 x 315 pixels
  • Minimum Size- 400 x 150 pixels
  • Perfect Size- Less than 100 KB

Few Tips On Size

  • If you want to load it fast, then use the recommended size.
  • After uploading your picture, do not drag the image.
  • Always upload PNG or JPEG files to ignore distortion.
  • Only upload your picture in a PNG file if your image has text or logos on it.

Facebook Timeline and Post Image Sizes

  • Minimum File Size- 600 x 315 pixels
  • Recommended File Size- 1200 x 630 pixels

Few Tips On Size

  • When you upload 5-10 pictures using the carousel display, you must upload them at 1200 x 1200 pixels.
  • The aspect ratio, in this case, must be 1:1.

Event Cover Photo Size of Facebook

  • Recommended Size- 1200 x 628 pixels

Few Tips

  • Always upload the image of the Facebook event at a 2:1 ratio.
  • Remember, you can not change anything after you add to the event.

Facebook Stories Picture Size

  • Recommended Size- 1080 x 1920 pixels

Few Tips

  • You must not upload the Facebook stories image below 500 pixels.
  • You must maintain a 9:16 aspect ratio.

Facebook Ads Picture Sizes

You need to get the perfect picture size for Facebook ads as well.

Facebook Feed Ads Picture Sizes

  • Minimum Size- 600 x 600 pixels
  • Maximum File Size- 30 MB
  • Recommended Size- 1080 x 1080 pixels

Sizes For Instant Article Pictures

  • Maximum File Size- 30 MB
  • Recommended Size- 1080 x 1080 pixels
  • Aspect Ratio- 91:1 to 1:1

Sizes For Marketplace Ads

  • Maximum File Size- 30 MB
  • Recommended Size- 1080 x 1080 pixels
  • Aspect Ratio- 1:1

Sizes For Facebook Right Column Ads

  • Recommended Size- 1080 x 1080 pixels
  • Minimum Size- 254 x 133 pixels
  • Aspect Ratio- 1:1

Sizes For Messenger Inbox Ads

  • Maximum File Size- 30 MB
  • Minimum Size- 254 x 133 pixels
  • Aspect Ratio- 1:1
  • Recommended Size- 1080 x 1080 pixels

Sizes For Facebook Search Images

  • Maximum File Size- 30 MB
  • Minimum Size- 600 x 600 pixels
  • Aspect Ratio- 91:1 to 1:1
  • Recommended Size- 1080 x 1080 pixels

Sizes For Messenger Stories Ads

  • Recommended Size- 1080 x 1080 pixels
  • Minimum Width- 500 pixels
  • Aspect Ratio- 9:16

Sizes For Sponsored Messages Pictures

  • Maximum File Size- 30 MB
  • Recommended Size- 1080 x 1080 pixels
  • Aspect Ratio- 91:1 to 1:1

Instagram Image Sizes

Instagram is a visual-based network where digital marketers can showcase their video and photography skills. You can reach your potential customers if you pay attention to static images. However, you need to know the correct dimensions, proper format, and size to make a significant difference.

Sizes For Instagram Profile Picture

You must display your profile picture on Instagram at 110 x 100 pixels. On the other hand, it should be stored at 320 x 320 pixels.

Sizes For the Instagram Feed Posts

  • Landscape– 1080 x 566 pixels
  • Square– 1080 x 1080 pixels
  • Portrait– 1080 x 1350 pixels
  • Aspect Ratio– From 1.91:1 to 4:5
  • Suggested Image Size– The Width of the picture must be 1080, whereas the height must be between 566 and 1350 pixels.

Few Valuable Tips

  • To get clear and sharp Instagram images, you must upload the picture at a width of 1080 at least.
  • If your picture resolution is lower than 320 pixels, you do not need to resize it, Instagram will do that for you.

Sizes For Instagram Photo Thumbnail

  • Suggested Upload Size- 1080 pixels
  • Display Size- 161 x 161 pixels

A valuable tip is that you can upload mainly large pictures to avoid pixelation.

Sizes For Instagram Stories Picture

  • Suggested Size- 1080 x 1920 pixels

Few Tips

  • You must upload your IG stories at an aspect ratio of 9:16. Otherwise, your story will be unnecessarily zoomed or cropped.
  • Try to upload your picture in a smaller size, keeping the aspect ratio the same to make your story buffer faster.

Sizes For Instagram Ads Images

  • Landscape– 1080 x 566 pixels
  • Square- 1080 x 1080 pixels
  • Maximum Width– 1080 pixels
  • Minimum Width– 320 pixels
  • Aspect Ratio– From 1.91:1 to 4:5.

Sizes For Instagram Reels

When you upload reels on Instagram, the cover photo must be 1080 x 1920 pixels.

Note-Instagram reels mainly have a 9:16 aspect ratio. It generally crops your spins to 1:1 in the profile feed and 4:5 in the home feed.

Sizes For Instagram Stories Ads

  • Suggested Size- 1080 x 1920 pixels

Sizes For Twitter Image

Gone are the days when controversial hot tech posts were all you had to garner a massive following on Twitter. Now in the Twitter world, you have to create or post interesting and eye-catching pictures to get attention from Twitter users and stand out from the rest.

If your brand account is on Twitter, upload different profiles and cover pictures. However, to get retweets, you need to know the right dimension to improve engagement on Twitter.

A Small Tip– For brands, it will be better if you upload educative and easily readable infographics to get massive attention from Twitter users.

Sizes For Twitter Profile Picture

  • Maximum File Size- 2 MB
  • Minimum Size- 200 x 200 pixels
  • Suggested Size- 400 x 400 pixels

Sizes For Twitter Header Picture

  • Recommended Size- 1500 x 1500 pixels

Few Tips

  • Always upload Twitter header pictures at the maximum size to cope with future changes.
  • The Twitter header image shows very differently on the browser and monitor; you can check your picture using both.

Sizes For Twitter Card Image

  • Supported Formats- PNG, JPG, and GIF
  • Maximum File Size- 1 MB
  • Minimum Size- 120 x 120 pixels

Sizes For Twitter In-stream Pictures

  • Maximum File Size– At least 5 MB for mobile and 15 MB for the web.
  • Minimum Size- 600 x 335 pixels
  • Suggested Size-1600 X 900 pixels
  • Supported Formats– GIF, PNG, and JPEG
  • Suggested Aspect Ratio– For desktop, it should be between 2:1 and 1:1. For mobile, it should be between 2:1, 3:4, and 16:9.

Sizes For Twitter Ads

  • For Single and multi-image tweets– the minimum size must be 600 x 335 pixels. Try uploading it in a larger size to get the best results.
  • For Website Card Pictures– the maximum file size should be 3 MB. When you maintain 800 x 418 pixels, the aspect ratio must be at 1.91:1. For 800 x 800 pixels, you should keep the aspect ratio of 1:1.

Note: For the sizes for App Card images, Conversion Card images, and Direct Message Card images, try to maintain the 800 x 418 pixels for a 1.91:1 aspect ratio. For 800 x 800 pixels, it must be a 1:1 aspect ratio.

Sizes For LinkedIn Images

LinkedIn is one of the most extensive professional networks existed on the internet. This platform is a great way to reach industry leaders, making it the right platform for B2B and recruiters. So, your responsibility is to highlight the most relevant things about your brand or business to grab the users attention.

You must produce interesting photos to enhance the possibility of getting noticed by so many people. Hence, it would help if you worked hard on the cover image and banner image of your LinkedIn profile. However, please stick to the suggested or recommended image sizes mentioned below to get desired results.

Pro-tip– Before you finalize your picture size, check if your picture looks good on every device or screen size.

Sizes For LinkedIn Personal Profile

Suggested image size For Profile Picture– 400 x 400 pixels

Suggested image size for Cover Picture- 1584 x 396 pixels

Suggested Aspect Ratio– 4:1

Sizes For LinkedIn Company Page Image

  • Company Logo- 300 x 300 pixels
  • Cover image– 1128 x 191 pixels
  • Square Logo– 60 x 60 pixels
  • Blog Post Picture-1350 x 440 pixels
  • Life Tab Brand Image– 900 x 600 pixels
  • Life Tab Custom Modules– 502 x 282 pixels
  • Life Tab Primary Image– 1128 x 376 pixels

Few Tips

  • You must upload the pictures in PNG and JPEG formats.
  • Try to use a 1.91:1 aspect ratio.
  • The suggested file dimension is 1200 x 628 pixels.

Sizes For LinkedIn Ads Pictures

  • Sponsored content pictures– 1200 x 627 pixels
  • Size of Spotlight ads logo– 100 x 100 pixels
  • Company logo size for the LinkedIn ads– 100 x 100 pixels
  • Spotlights ads custom background picture– 300 x 250 pixels

Sizes For YouTube Image

YouTube has become a go-to platform for many people since many active users are already present there. Therefore, when it comes to YouTube, you must be well-versed in every little detail of this platform. An attractive profile and cover photo can be a game-changer for your channel.

So, whether it is about the sizes of your profile picture or banner image, they all require equal attention. You must work on the upload size to make your visual content more accurate and eye-catching.

  • Recommended Size for Profile Picture- 800 x 800 pixels
  • Suggested Size for Cover image- 2560 x 1440 pixels
  • Minimum size for the banner picture- 2048 x 1152 pixels
  • Maximum file size for banner image- 6 MB
  • Supported Aspect Ratio for banner image- 16:9

Few Tips

  • You must upload your profile picture in PNG, JPEG, BMP, or GIF formats. Though, animated GIFs will not be rendered.
  • You must ensure you are using the updated version to upload large files more significant than 20 MB.

Sizes For YouTube Videos

  • Minimum file size– 1280 x 720 pixels
  • Suggested aspect ratio– 16:9
  • For 2k videos– 2560 x 1440 pixels
  • For 4k videos– 3840 x 2160 pixels
  • 144P- 256 x 144 pixels
  • 240P– 426 x 240 pixels
  • 360P- 640 x 360 pixels
  • 480P– 854 x 480 pixels
  • 720P– 1280 x 720 pixels
  • 1080P– 1920 X 1080 pixels

Image Sizes For Snapchat

Snapchat is a platform solely for entertainment purposes, where people or brands share stories in an exciting way. There are several filters as well to make everything more fun. So, you need to know the correct dimension before you upload your content on Snapchat.

Image Sizes For Snapchat Ads

  • File format- PNG or JPG
  • Minimum size- 1080 x 1920 pixels
  • Maximum file size- 5 MB
  • Aspect Ratio- 9:16

Image Sizes For Snapchat Geofilter

  • Maximum file size- 5 MB
  • Minimum size- 1080 x 1920 pixels
  • File type- PNG or JPG
  • Aspect Ratio- 9:16

Conclusion

A plethora of things is there that social media marketers have to consider while producing videos and images on social media platforms. Among them, getting the proper dimension and image sizes is crucial to keep your audience engaged.

Accurate image sizes not only improve your audience engagement but also helps you to ignore weird display of pictures and pixelation. Therefore, we have provided a list of all image sizes for different social media platforms to help you reach your maximum potential.
